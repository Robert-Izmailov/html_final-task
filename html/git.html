Git

  • git init — "создать новый локальный репозиторий".
  • git clone — "скачать существующий проект с сервера (удаленный репозиторий)".
  • git add <имя_файла> или git add . — добавить файлы в индекс для подготовки к коммиту.
  • git commit -m "Сообщение" — сохранить изменения (коммит) в локальном репозитории.
  • git status — проверить текущее состояние (измененные или новые файлы).
  • git log — просмотреть историю всех коммитов.
    • git pull — скачать и применить изменения с сервера в вашу текущую.
  • git push -u origin <имя_ветки> — загрузить локальные коммиты на сервер.
